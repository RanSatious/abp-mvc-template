# 前端编码规范

## 编辑器

vscode

## 插件

- HTML Snippets
- Prettier
- ESLint
- gitlens

## vscode 设置

```json
{
  "editor.formatOnSave": true,
  "editor.tabSize": 2,
  "prettier.singleQuote": true,
  "prettier.printWidth": 160,
  "html.format.wrapAttributes": "auto",
  "editor.renderWhitespace": "all",
  "editor.renderControlCharacters": true,
  "editor.detectIndentation": false
}
```
